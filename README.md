# 🏢 StaffCore EMS: Enterprise Edition

StaffCore is a robust, data-driven **Employee Management System** built with **Python 3.13**, **Streamlit**, and **Plotly**. It provides HR administrators with a "Command Center" to manage personnel data and visualize financial distributions across departments.

## 🌟 Key Features
- **Executive Dashboard**: Real-time metrics tracking total staff, monthly payroll, and departmental counts.
- **Relational SQL Persistence**: All records are stored in a persistent SQLite database with unique ID verification.
- **Interactive Analytics**: High-fidelity Plotly charts showing salary distribution with custom color gradients.
- **Record Lifecycle Management**: Full CRUD functionality allowing HR to register, search, and securely remove staff records.
- **Luxury UI Design**: Dark-themed glassmorphism interface using custom CSS injections and Unsplash-backed overlays.

## 🛠️ Tech Stack
- **Backend**: Python 3.13
- **Database**: SQLite3
- **Visualization**: Plotly Express
- **Frontend**: Streamlit (Advanced CSS & Markdown)
- **Data Handling**: Pandas
